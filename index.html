CSS Selectors and Specificity

Selector Example Description
Element Selector p { color: blue; } Selects all <p> elements.
Class Selector .example { background-color: yellow; } Selects all elements with class="example".
ID Selector #unique { font-weight: bold; } Selects the element with id="unique".
Pseudo-class Selector a:hover { text-decoration: underline; } Selects links on mouse over.
